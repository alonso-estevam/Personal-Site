Olá! Este é o meu primeiro exercício prático do curso "Introdução a criação de websites com HTML5 e CSS3", do bootcamp da MRV, disponível na plataforma da Digital Inovation One. No curso aprendi a estrutura básica de um site, criando um arquivo index e adicionando elementos como o !DOCTYPE html, head e body. Também aprendi algumas tags semânticas do HTML, como o article, header e footer; como fazer listas ordenadas e não ordenadas; como criar links e adicionar imagens.

Além de elementos HTML, o curso abordou a estilização com CSS. Aprendi que as CSS funcionam a partir de "Box Model", ou modelo em caixa. Box model é a forma como o navegador representa cada elemento HTML, e ele tem as seguintes propriedades: margin, border, padding e content. Enquanto a HTML fornece o conteúdo e a semântica, as CSS nos permitem estilizar o conteúdo de um site. Nesse sentido, há duas maneiras de identificar um grupo de elementos para aplicar estilos: o ID (através do ponto [.]) e as classes (através da "hashtag" [#]). O id é único, já as classes podem se repetir.
